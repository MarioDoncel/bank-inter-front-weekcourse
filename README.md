## Getting Started

### Prerequisites

<!-- This is an example of how to list things you need to use the software and how to install them. -->
* npm
  ```sh
  npm install npm@latest -g
  ```
* PostgreSQL

### Installation

<!-- _Below is an example of how you can instruct your audience on installing and setting up your app. This template doesn't rely on any external dependencies or services._
 -->

1. Clone the repo
   ```sh
   git clone https://github.com/MarioDoncel/bank-inter-front-weekcourse
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the Backend  at <a href="https://github.com/MarioDoncel/bank-inter-back-weekcourse">https://github.com/MarioDoncel/bank-inter-back-weekcourse</a>

    
4. Run the application
    ```js
    npm start
    ```
